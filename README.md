# Project Physica

Project Physica is a collection of projects that include nuclear research, quantum computing, programming and more! We have 3 categories of project currently: Quantum, PQAR, and Programming.

Our Quantum Branch is where we have projects related to quantum research. We currently are building an ion trap quantum computer for testing. Our Quantum Branch is also a branch of PQAR.

Our PQAR projects are a set of project directly related to particle physics and theoretical physics. (PQAR stands for Plex Quantum and Atomic Research).

Our Programming Branch does various programming activities. One current project of our Programming Branch is to create a new Linux (Debian) based Operating System specifically for projects like the ones in PQAR. We have done lots of nuclear research using simulations and databases in this branch. Currently a partner of Project Physica is making a programming language that will be used for controlling Qiskit in our Quantum Computer(s).
